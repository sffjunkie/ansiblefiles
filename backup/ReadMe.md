# Backup

Backs up data from

* Laptop
    * music, pictures etc.
* Servers
    * Service configuration
    * Service data

## Targets

* OneDrive - Via TrueNAS NFS mount or rclone
* Google Drive - Via TrueNAS NFS mount or rclone
* TrueNAS - NFS mount
* Offsite - Backup server
* USB HD

* Encrypt before uploading
